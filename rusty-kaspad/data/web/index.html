Support the Network

The very heart of Kaspa is the public nodes. Please consider making your node public.

To support the network, forward TCP port 16111 on your firewall to your Umbrel node IP address.

About Rusty Kaspad

Rusty Kaspad is a Rust implementation of a Kaspa full node. Kaspa is a proof-of-work cryptocurrency with instant transaction confirmation through blockDAG technology.

This containerised version of Rusty Kaspad was built by Supertypo based on the official Rusty Kaspad source code, and packaged for Umbrel by Luke Dunshea.
